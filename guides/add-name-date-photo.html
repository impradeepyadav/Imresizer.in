  • Photo should be recent (within 3 months)

3

Add Your Name

Enter your name exactly as in your SSC application:

  • Text: Your full name (as per 10th certificate)
  • Case: UPPERCASE (recommended)
  • Font size: 10-12px (adjust to fit)
  • Color: Black (most visible)
📌 Name Format: Use exactly as registered. If your name is "Rahul Kumar", write "RAHUL KUMAR" not "Rahul" only.

[Screenshot: Name entered in text field]

✏️ Enter your full name in uppercase

4

Add Date in Correct Format

This is the most critical part - date must be in DD/MM/YYYY format only:

✅ CORRECT FORMAT:

  • 15/03/2026 (with slashes)
  • Day: 01-31 (two digits)
  • Month: 01-12 (two digits)
  • Year: 2026 (four digits)

❌ INCORRECT FORMATS:

  • 15-03-2026 (dashes instead of slashes)
  • 15.03.2026 (dots instead of slashes)
  • 15/3/2026 (month single digit)
  • 15/03/26 (year two digits)
  • 2026/03/15 (wrong order)

Text settings:

  • Font size: 8-10px (slightly smaller than name)
  • Color: Black
  • Position: Below name

[Screenshot: Date entered as DD/MM/YYYY]

📅 Date must be DD/MM/YYYY with slashes

5

Position Text Correctly

SSC requires text at the bottom of the photo:

  1. Position name and date at the bottom edge
  2. Name above, date below (or both in one line)
  3. Leave small margin from bottom
  4. Center aligned horizontally
🎯 Positioning Tip: Text should be inside the photo area, not outside. Don't place it too close to face.

Preview and Adjust

Before finalizing:

  • Check spelling of your name
  • Verify date format (DD/MM/YYYY)
  • Ensure text is readable (not too small)
  • Confirm text doesn't cover your face

  2. Download the image
  3. Open and verify name and date are visible
  4. Check file size (use compressor if needed)

📋 SSC-Specific Requirements Summary

Requirement Specification
Photo Dimensions 4.0 cm x 2.0 cm (520 x 260 pixels at 300 DPI)
Name Text Full name as per application, UPPERCASE recommended
Date Format DD/MM/YYYY only (example: 15/03/2026)
Text Position Bottom of photo, centered
File Size 20 KB to 50 KB (use compressor after adding text)
Format JPG only

✅ Common Mistakes to Avoid

  • Wrong date format: Using dashes (15-03-2026) instead of slashes (15/03/2026)
  • Incomplete name: Writing only first name, not full name
  • Wrong position: Text at top or covering face
  • Two-digit year: Using 26 instead of 2026
  • Text too small: Not readable when photo is viewed at actual size
  • Wrong date: Using application date instead of photo taken date

📅 Which Date to Use?

Use the date when the photo was taken. For example, if you took the photo on 15th March 2026, write: 15/03/2026

Not the application date, not your birth date - the date the photo was captured.

❓ Frequently Asked Questions

Q: Is name and date on photo mandatory for all SSC exams?

A: Yes, for SSC CGL, CHSL, MTS, and most SSC exams. Some posts may have different requirements - always check your specific notification.

Q: What if my name is very long?

A: Use a slightly smaller font size, but ensure it's readable. You can also use initials for middle names if needed.

Q: Can I use dd-mm-yyyy format?

A: No. SSC strictly requires DD/MM/YYYY with slashes. Using dashes or dots may lead to rejection.

Q: Where exactly should the text be placed?

A> At the bottom of the photo, centered. Not on the face, not outside the photo area.

Q: What font and size should I use?

A> Use a simple, readable font like Arial. Size around 10-12px for name, 8-10px for date. Adjust based on your photo.

Q: Can I add name and date using my phone?

A> Yes, our tool works on all devices. You can do it entirely on your phone.

Q: After adding text, my file size increased. What to do?

A> Use our Image Compressor to reduce to 20-50KB while maintaining quality.
