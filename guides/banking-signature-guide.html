How to Create a Signature for Banking Exams

📅 Updated: March 2026 • 📁 Banking • ⏱ 7 min read

Specific requirements for IBPS, SBI, RRB signature uploads. Size, format, and tips for acceptance. Banking exams are very strict about signature specifications - one wrong pixel can lead to rejection.

Critical: Banking Signature Requirements

Banking exams (IBPS, SBI, RRB) have the strictest signature requirements. File size must be between 10-20KB only. Even 21KB will be rejected. Format must be JPG only.

❌ Top 5 Reasons for Signature Rejection in Banking Exams

  1. File size out of range (10-20KB): Most common reason - file too large or too small
  2. Wrong dimensions: Not exactly 4.0 x 2.0 cm
  3. PNG format instead of JPG: Banking portals accept only JPG
  4. Signature too small or too light: Not clearly visible
  5. Wrong background color: Grey or colored instead of white

💡 Pro Tips for Banking Signatures

  • Practice drawing: If using draw method, practice 2-3 times to get smooth signature
  • Save at higher quality first: Create signature at good quality, then compress to target size
  • Check on different screens: Signature should be visible on both mobile and computer
  • Keep consistent: Use same signature for all banking applications (IBPS, SBI, RRB)
  • Take screenshot of file properties: Before final submission, screenshot showing correct size and format

❓ Frequently Asked Questions

Q: What is the exact signature size for IBPS PO?

A: IBPS PO requires signature of 4.0 cm x 2.0 cm (width x height). File size must be 10-20KB in JPG format.

Q: My signature is 25KB. How do I reduce it to 15KB?

A: Use our Image Compressor and set target size to 15KB. The tool will automatically adjust quality.

Q: Can I use blue ink for my signature?

A: No. Banking exams require black ink on white background only. Blue ink may not scan well.

Q: What if my signature is rejected?

A: Most rejections are due to file size. Recompress to exactly 15KB. Also verify dimensions are 4.0 x 2.0 cm.

Q: Is PNG format accepted for banking exams?

A: No. Banking portals accept only JPG/JPEG format. Convert PNG to JPG before uploading.

Q: Can I use the same signature for all banking exams?

A: Yes! IBPS, SBI, RRB all have same requirements. Create once, use for all.

Q: Should I sign in cursive or print?

A: Cursive (running hand) is preferred for banking exams. It looks more authentic and professional.
