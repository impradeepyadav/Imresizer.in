# ImResizer.in – Free Online Image Resizing & Compression Tools

[ImResizer.in](https://imresizer.in) is a free, privacy-focused online toolkit designed to help students, professionals, and everyday users resize, compress, and convert images for exams, passports, visas, signatures, and official documents. All tools run entirely in your browser – **no uploads, no servers, no data collection**.

## 🚀 Live Website

Visit the live application: [https://imresizer.in/](https://imresizer.in/)

## ✨ Key Features

ImResizer provides a suite of powerful tools, all with an intuitive interface and precise controls.

### 🖼️ Image Compressor & Resizer
*   Compress JPG, PNG, and WEBP images to **exact file sizes** like 20KB, 50KB, or 100KB.
*   Resize images in **pixels, centimeters, or inches** (with 300 DPI for print quality).
*   Maintain aspect ratio, crop, rotate, and flip images.
*   Apply filters like Grayscale and Signature Mode.
*   Add **Name & Date overlay** in DD/MM/YYYY format for SSC and other government exam forms.

### 🛂 Passport & Visa Photo Maker
*   Pre-loaded presets for **50+ countries** (USA, UK, India, Canada, China, Schengen, etc.).
*   Change background to white, light blue, light gray, or any custom color.
*   An on-canvas **head-size guide circle** for perfect positioning.
*   Print multiple photos on a single **4×6 or A4 sheet**.

### ✍️ Digital Signature Maker
*   Create signatures using three methods: **Draw** (mouse/touch), **Type** (various fonts), or **Upload**.
*   Adjust pen size, color, and background.
*   Download as **PNG (transparent background)** or JPG.

### 📄 PDF Converter
*   **Image to PDF:** Combine multiple images into one PDF (A4, Letter, Legal, or fit to image).
*   **Merge PDF:** Combine several PDF files into a single document.
*   **PDF to Images:** Extract each page of a PDF as a JPG or PNG file.

## 🔒 Privacy First

*   **100% Client-Side:** All processing happens in your browser using JavaScript.
*   **No Uploads:** Your files never leave your device. We have no servers to receive your data.
*   **No Tracking:** We do not collect personal information or store your images.

## 🛠️ Built With

*   **HTML5 / CSS3** – Custom styling with CSS variables for a consistent theme.
*   **JavaScript (ES6)** – Core tool logic and interactivity.
*   **Lucide Icons** – Clean, consistent icon set.
*   **Cropper.js** – Advanced image cropping functionality.
*   **jsPDF** – Client-side PDF generation.
*   **PDF.js** – Parsing and rendering PDF files for image extraction.
